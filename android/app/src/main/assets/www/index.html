외부 접속 방법 (localhost)

1 ngrok 회원가입 및 토큰 받기

인터넷 브라우저로 ngrok 공식 홈페이지(ngrok.com)에 접속합니다.

[Sign up for free]를 눌러 무료 회원가입을 진행합니다 (구글 계정 연동 가입 시 5초 완료)

로그인 후 대시보드 화면 중간의 [Your Authtoken] 메뉴를 누르면, 사용자님 고유의 보안 인증 토큰이 보입니다 (예: 2abc...). 이 값을 복사합니다.

2 파워쉘에서 토큰 등록 & 터널 시작

컴퓨터 화면 윈도우 검색창에 powershell을 검색하여 실행합니다.

파란 창에 복사한 토큰을 저장하기 위해 아래 명령어를 복사/붙여넣고 엔터를 입력합니다:

npx ngrok config add-authtoken <여기에 복사한 토큰 붙여넣기>

(예: npx ngrok config add-authtoken 2abcedfg12345...)

등록 완료 후, 아래 명령어를 실행하여 3333번 포트를 외부로 개방합니다:

npx ngrok http 3333

3 화면에 뜬 주소 복사하여 스마트폰에 입력

명령어가 가동되면 파워쉘에 검은 인터페이스 창이 뜨며 아래와 같은 연결 경로가 활성화됩니다:

Forwarding   https://a1b2-c3d4.ngrok-free.app -> http://localhost:3333

여기서 생성된 https://a1b2-c3d4.ngrok-free.app 형식의 주소를 스마트폰 MyDays 앱의 [설정] -> [서버 주소] 칸에 입력한 뒤 [설정 저장하기] 버튼을 누르면 모든 세팅이 완료됩니다!

사각 말풍선 도움말

각 PHOTO(사진) 위에 네모난 사각 말풍선 상자를 만들고, 그 안에 어울리는 소제목 글을 자동으로 넣어 강조해 주는 기능입니다.

✍️

60자 설명문 도움말

각 PHOTO(사진) 바로 위에 사진을 설명하는 약 60자 내외의 설명 글(본문 내용)을 자동으로 생성하여 본문을 풍성하게 만들어 주는 기능입니다.
